# Print a **tblr** table object

This function uses `tblr_as_latex` to generate LaTeX markup from the
`tblr` table object and prints it to the console.

## Usage

``` r
# S3 method for class 'tblr'
print(x, ...)
```

## Arguments

- x:

  A `tblr` table object.

- ...:

  Any additional parameters.

## Value

A character string containing LaTeX markup.
